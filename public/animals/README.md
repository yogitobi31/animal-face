# Animal illustration assets

- Place production illustration files in this folder.
- Use `kebab-case` filenames and `.webp` format.
- Example names:
  - `proud-cat.webp`
  - `sunny-dog.webp`
  - `snow-arctic-fox.webp`
  - `moon-fox-cat.webp`

When a file is added with a matching `illustrationKey`, result cards will render it automatically.
